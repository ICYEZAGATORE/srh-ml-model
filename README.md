# srh-ml-model

**AI-Powered SRH Education Platform вЂ” ML Model Repository**  

## Project Description

This repository contains the machine learning pipeline for an inclusive, AI-powered sexual and reproductive health (SRH) education platform designed for Rwandan teenagers (13вЂ“19) and persons with disabilities. The platform uses a **Retrieval-Augmented Generation (RAG)** architecture with a bilingual conversational agent supporting **English and Kinyarwanda**, and trains three classifiers from scratch to handle safety moderation, language detection, and topic routing.

Research grounded in WHO, UNFPA, UNESCO ITGSE, and Rwanda Ministry of Health guidance, with target deployment in Kigali City and Eastern Province.

## What This Repo Will Train

Three classifiers built from scratch + a RAG pipeline. 

| Model | What it does | Algorithms compared | Output |
|---|---|---|---|
| **Model 1 вЂ” Safety classifier** | Binary: blocks harmful queries (sexual violence, self-harm, age-inappropriate) | Logistic Regression, SVM, XGBoost, MLP | `safety_classifier.joblib` |
| **Model 2 вЂ” Language detector** | EN vs Kinyarwanda routing | Logistic Regression, Naive Bayes, SVM | `language_detector.joblib` |
| **Model 3 вЂ” Topic classifier** | Multi-class SRH topic routing (contraception, STIs, consent, GBV, disability-SRH, вЂ¦) | Decision Tree, SVM, XGBoost, MLP | `topic_classifier.joblib` |

Multilingual sentence-transformer embeddings (`paraphrase-multilingual-MiniLM-L12-v2`) and the LLM generator (Gemini 2.0 Flash by default, swappable to GPT-4o) are pre-trained components вЂ” not retrained вЂ” which is standard practice.
